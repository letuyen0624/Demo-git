Workspace Git Demo

Trang web kiểm tra trạng thái thay đổi mã nguồn

Trạng thái hiện tại
user@git-demo:~$ git status
On branch main
Untracked files:
  (use "git add <file>..." to include in what will be committed)
      index.html
      tuyen.txt
Các bước để kiểm tra thay đổi
1

Tạo file mới

Tạo thêm bất kỳ file nào khác trong thư mục này (ví dụ: test.html hoặc style.css) để kiểm tra cơ chế Untracked của Git.

2

Kiểm tra git status

Mở terminal trong thư mục này và gõ lệnh git status để thấy danh sách các file mới vừa được tạo.

3

Theo dõi file (Track file)

Dùng lệnh git add . hoặc git add index.html để chuyển các file mới sang trạng thái chuẩn bị commit (Staged).
